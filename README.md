## Super Easy Install

1. Download the repo

```sh
https://github.com/carolina-loaiza/FrontEndTest.git
```

2. Install dependencies

```sh
npm install
```

3. Run gulp task to local server

```sh
gulp server
```

3. Go to localhost in your browser

```sh
http://localhost:8080/
```
